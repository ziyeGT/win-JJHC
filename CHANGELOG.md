# Huaci 更新记录

## 1.3.2 — 译、框、记、飞

发布日期：2026-07-17

从 `1.0.1` 到 `1.3.2`，Huaci 已经不只是一个“划一下就翻译”的小工具，而是一个仍然保持极简外观的 Windows 桌面效率盒子：划词、手动互译、截图 OCR、快速笔记和本地闹铃都集中在一个 256×162 的小窗口里。

### 这版最值得玩的地方

- **划一下就出答案**：鼠标选中英文后，在原位置附近出现灰黑色翻译 Toast；鼠标移开立即消失，需要多看一会儿就点钉子。
- **截图里的文字可以“原地变中文”**：按住 `Ctrl + Alt + 鼠标左键` 直接拖框，松开后执行本地 OCR 和翻译。结果不是另开一个普通弹窗，而是在原选区上方重新排版，保留段落、页边距和大致字号，并能随时切回原图。
- **闹铃不是普通通知框**：时间到了，一架由 WPF 矢量路径绘制的飞机会拖着文字横幅从屏幕上方循环飞过，直到手动关闭。
- **飞机不是一张静态图片**：机翼会轻微起伏，机身有俯仰和上下浮动，引擎风扇会转，航行灯会闪，牵引线和横幅也有自己的动态。
- **还有飞越声音**：提示音由程序在内存中实时合成，不需要额外携带音频文件；同一个闹铃只播放一次，不会随着飞机循环反复轰炸耳朵。
- **一个“译”字走天下**：EXE、桌面快捷方式、系统托盘和主页面划词入口统一使用“译”字标识。

### 新增：离线中英互译

- 内置 Mozilla Firefox Translations / Bergamot WASM 翻译引擎。
- 同时内置 `英语 → 简体中文` 与 `简体中文 → 英语` 两套模型。
- 默认使用“离线优先”，没有网络也能完成常用英中互译。
- 手动翻译窗口加入方向切换按钮；切换方向时，已有原文和译文可自动交换。
- 保留“仅离线”“离线优先”“仅在线”三种翻译路线。
- 在线模式兼容 OpenAI 风格接口，API Key 只进入 Windows Credential Manager，不写入设置文件。

### 新增：截图 OCR 翻译

- 默认快捷手势：`Ctrl + Alt + 鼠标左键拖动`。
- 支持多显示器和不同 DPI 缩放。
- 框选阶段显示清晰选框，避免不知道自己截了哪里。
- 松开鼠标后选框不会消失；识别、翻译和结果展示均在该区域继续完成。
- 内置 RapidOcrNet 与 PP-OCRv5 mobile 中英文识别模型。
- OCR 行坐标用于重建段落、页边距、字号和阅读顺序。
- 结果支持：
  - 原图 / 译文切换
  - 一键复制译文
  - 手动关闭
- OCR 结果不会因鼠标移开或超时自动消失。
- 截图像素只在本机内存中处理，Huaci 不会把截图写成临时文件。

### 新增：快速笔记

- 支持直接输入文字并保存。
- 支持 `Ctrl+V`、按钮粘贴和拖放图片。
- 可从资源管理器复制图片文件，也兼容部分应用发布的旧式剪贴板位图。
- 文字保存为 UTF-8 `.txt`，图片保存为 `.png`。
- 历史记录支持浏览、复制和删除。
- 默认保存到程序旁的 `save` 文件夹；目录不可写时自动回退到 `%LocalAppData%\Huaci\save`。
- 笔记只保存在本机，不会被自动翻译或上传。

### 新增：本地闹铃

- 可直接使用编辑框文字设置提醒，也能从已有文字笔记创建。
- 支持指定日期、小时、分钟，以及 10 分钟、30 分钟、1 小时后的快捷时间。
- 闹铃与普通笔记共用历史列表，不再分成两个面板。
- 闹铃条目带时钟标识，点击可查看准确时间或取消。
- 闹铃数据保存到 `save\alarms.json`。
- 成功设置闹铃后自动清空笔记输入框；取消设置则保留原内容。
- 多个闹铃同时到点时按时间顺序逐个展示。
- 程序重启后，尚未确认的过期闹铃仍会继续提醒。

### 主窗口与交互升级

- 主窗口缩小为 256×162，保持灰黑半透明极简风格。
- 首页只保留“划词、翻译、截图、笔记、设置”五个图标。
- 每行最多 4 个图标，多余模块自动从下一行左侧继续排列。
- 图标点击后不再残留蓝色键盘焦点框。
- 程序启动后默认显示主窗口，并默认开启自动划词。
- 主窗口默认常驻最上层；划词和截图不会再让它躲到其他应用后面。
- 只有用户点击最小化、X，或按下 `Ctrl+F1` 时才隐藏到托盘。
- 最小化与 X 使用同一套托盘隐藏逻辑，不会退出后台功能。
- 新增全局 `Ctrl+F1` 快速显示 / 隐藏主窗口。
- 如果快捷键在启动瞬间被其他程序占用，Huaci 会每 4 秒自动重试，成功恢复后给出通知。
- 新增随 Windows 开机启动选项，默认关闭且不需要管理员权限。

### Toast 和设置窗口优化

- 普通划词 Toast 去掉外层白框，只保留半透明灰黑主体。
- 鼠标移出 Toast 后立即消失。
- 固定后忽略鼠标移开和超时；取消固定后恢复正常消失逻辑。
- 长译文可滚动，不会把小窗口无限撑大。
- 设置内容集中到独立、可滚动的小窗口中。
- 自动划词开关只由首页“划词”图标控制，设置页只保留兼容模式等详细选项。
- `Ctrl+C` 兼容取词模式默认关闭，按需开启。

### 重点修复

- 修复主窗口在划词或截图后看似“消失”的问题：实际原因是置顶状态被撤销，窗口被浏览器等应用遮住。
- 修复点击截图入口后主窗口尺寸、位置或状态发生变化的问题。
- 修复 `Ctrl+F1` 偶尔第一次按下没有召唤窗口的问题。
- 修复系统热键启动注册失败后整次运行都无法恢复的问题。
- 修复 OCR 框选松手后选框立即消失的问题。
- 修复 OCR 结果错误套用普通 Toast“鼠标移开即关闭”逻辑的问题。
- 修复快速笔记无法正确粘贴部分图片格式的问题。
- 修复设置闹铃后输入文字仍留在编辑框的问题。
- 修复快速笔记历史刷新偶尔从后台线程更新 WPF 控件的问题。
- 修复点击模块后残留蓝色焦点边框的问题。
- 修复高 DPI 环境下部分窗口内容显示不完整的问题。

### 隐私与便携

- 默认优先本地翻译，OCR 始终在本机执行。
- API Key 不会写入 `settings.json`。
- 便携 ZIP 自带 .NET 运行时、双向翻译模型和 OCR 模型。
- 无需安装程序，解压后直接运行 `Huaci.exe`。
- 当前便携 ZIP 约 173 MiB，解压后约 333 MiB。
- 程序仍复用系统 Microsoft Edge WebView2 Runtime；Windows 11 通常已经自带。

### 升级与使用提示

1. 更新前请先从系统托盘菜单彻底退出旧版；点击主窗口的 X 只会隐藏程序。
2. 完整解压 `Huaci-1.3.2-win-x64-portable.zip` 后再运行，不要直接在 ZIP 压缩包内双击 EXE。
3. 便携版无需预装 .NET，也不要求管理员权限。
4. `%LocalAppData%\Huaci` 中的设置和 Windows Credential Manager 中的 API Key 会继续保留。
5. 如果旧笔记保存在旧版程序旁的 `save` 文件夹，请把该文件夹复制到新版目录。
6. 当前个人内测 EXE 尚未进行商业代码签名，Windows SmartScreen 可能显示“未知发布者”；确认文件来源和 SHA-256 后，可选择“更多信息 → 仍要运行”。
7. 系统仍需 Microsoft Edge WebView2 Runtime。

### 当前边界

- 内置离线翻译目前主要支持英语与简体中文，其他语言需要在线服务。
- 普通权限运行时，受 Windows UIPI 限制，无法读取或截取管理员权限窗口。
- 某些自绘软件不能向 UI Automation 提供选中文本，可尝试开启 `Ctrl+C` 兼容模式。
- OCR 精度仍会受低清图片、复杂背景、压缩噪声、遮挡和特殊字体影响。
- 截图“替换式翻译”是覆盖在原区域上方的本地阅读层，不会直接修改网页或文档内容。
